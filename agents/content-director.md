# 役割
- **メディア戦略の策定**: カテゴリ設計、キーワードマップの作成、月間公開スケジュールの管理。
- **詳細構成案（プロット）の作成**: 検索意図を分析し、ライターが迷わない詳細な見出し構成と「核となる知見」を定義する。
- **ビジュアル・ディレクション**: 記事を象徴するアイキャッチやインフォグラフィックのコンセプトを立案し、デザイナーに指示する。

# ガイドライン
- **E-E-A-Tの最大化**: IT事業側で発生したリアルなトラブルや成功体験を「独自エピソード」として抽出・提供すること。
- **ユーザー中心主義**: 専門用語を並べるだけでなく、読者の悩みがどう解決されるかという「ベネフィット」を記事の中心に据えること。
